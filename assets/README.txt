💖 HOW TO PERSONALIZE DEVUTTY'S BIRTHDAY WEBSITE 💖
====================================================

You only ever need to edit ONE file for words:  js/config.js
And drop your files into the two folders below.

------------------------------------------------
1) PHOTOS   ->   assets/photos/
------------------------------------------------
Put her photos here and name them:
   1.jpg  2.jpg  3.jpg  4.jpg  5.jpg  6.jpg
(For the Secret Vault:  secret1.jpg  secret2.jpg)

- .jpg / .png / .webp all work.
- If a photo is missing, a cute emoji placeholder shows
  automatically, so the site never looks broken.
- You can change filenames/captions in js/config.js -> gallery

------------------------------------------------
2) MUSIC    ->   assets/music/
------------------------------------------------
Add these audio files (any are optional):
   intro.mp3    -> soft piano for the opening + most of the site
   ending.mp3   -> emotional song for the surprise ending
   song.mp3     -> the song for the Music Corner vinyl player

- The site works fine with no music files. Even then, every
  button / popup still has a built-in chime (generated in the
  browser), so it never feels silent.
- Tip: press ♪ (bottom-right) to mute all sound, ◐ for dark mode.
- Browsers only allow sound AFTER the first tap/click, so the
  intro asks the visitor to tap — that's normal and expected.
- Use royalty-free / your own music. Good free sources:
  pixabay.com/music , incompetech.com , frepd.com

------------------------------------------------
3) WORDS / MESSAGES  ->  js/config.js
------------------------------------------------
Open js/config.js in any text editor. Everything is labeled:
  - intro lines
  - the long personal letter
  - timeline memories
  - secret vault answer + note
  - playlist + dedications
  - random cute messages
  - the surprise ending lines

------------------------------------------------
4) HOW TO OPEN IT
------------------------------------------------
Just double-click  index.html  to open in a browser.
Everything runs locally — no install needed.

Tip: to share it online for free, upload the whole folder to
Netlify (drag-and-drop at app.netlify.com/drop) or GitHub Pages.
